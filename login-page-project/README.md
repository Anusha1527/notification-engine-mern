# Login Page Project

This project is a simple login page application built with React and TypeScript. It demonstrates how to create a functional login form that displays mock credentials for demonstration purposes.

## Project Structure

```
login-page-project
├── src
│   ├── components
│   │   └── LoginPage.tsx
│   ├── mock
│   │   └── credentials.ts
│   ├── App.tsx
│   └── types
│       └── index.ts
├── public
│   └── index.html
├── package.json
├── tsconfig.json
└── README.md
```

## Features

- A functional login form that accepts user input.
- Displays mock credentials for user reference.
- Built with TypeScript for type safety.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/login-page-project.git
   ```

2. Navigate to the project directory:
   ```
   cd login-page-project
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.
